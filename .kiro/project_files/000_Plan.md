## Executive Summary

### The Goal

Build a **medical education platform** (HaffNet L4yerCak3) that displays CME courses and handles doctor registrations, using the **vc83-com l4yercak3 API backend** for all business logic, payment processing, and document generation.

### The Approach

- **Frontend**: Next.js 16 + React 19 + Tailwind 4 (shadcn/ui components)
- **Backend**: vc83-com API (Convex-based) via REST endpoints
- **Pattern**: External website integration (similar to value calculator)

### Key Understanding: Objects from API Populate UI

**IMPORTANT**: The frontend displays UI components that are **populated with objects from the API backend**:

- **API Objects â†’ UI Display**:
  - `objects` table with `type: "event"` â†’ Displayed as **Courses** in catalog
  - `objects` table with `type: "product"`, `subtype: "ticket"` â†’ Course offerings
  - `objects` table with `type: "transaction"` â†’ Created on registration
  - `objects` table with `type: "ticket"` â†’ Generated by behaviors after registration
  - `objects` table with `type: "invoice"` â†’ Created if employer pays

- **Behaviors handle all logic**, **UI consumes the results**
- Frontend fetches data via REST API endpoints
- Backend workflows process everything automatically

### API Client Implementation

```typescript
// lib/vc83-api/client.ts
const API_BASE_URL = process.env.NEXT_PUBLIC_VC83_API_URL!;
const API_KEY = process.env.VC83_API_KEY!;

class VC83APIClient {
  private async request<T>(endpoint: string, options?: RequestInit): Promise<T> {
    const url = `${API_BASE_URL}${endpoint}`;

    const response = await fetch(url, {
      ...options,
      headers: {
        'Authorization': `Bearer ${API_KEY}`,
        'Content-Type': 'application/json',
        'X-External-Domain': 'haffnet-l4yercak3.vercel.app',
        ...options?.headers,
      },
    });

    if (!response.ok) {
      const error = await response.text();
      throw new Error(`API Error: ${response.status} - ${error}`);
    }

    return response.json();
  }

  // Get all published courses (events)
  async getCourses(filters?: {
    specialty?: string;
    format?: string;
    status?: 'published';
  }) {
    const params = new URLSearchParams();
    if (filters?.status) params.append('status', filters.status);
    if (filters?.specialty) params.append('specialty', filters.specialty);
    if (filters?.format) params.append('format', filters.format);

    return this.request<{
      events: CourseEvent[];
      total: number;
    }>(`/api/v1/events?${params.toString()}`);
  }

  // Get specific course details
  async getCourseDetails(courseId: string) {
    return this.request<CourseEvent>(`/api/v1/products/${courseId}`);
  }

  // Trigger registration workflow
  async registerForCourse(data: RegistrationData) {
    return this.request<{
      success: boolean;
      transactionId: string;
      message: string;
    }>('/api/v1/workflows/trigger', {
      method: 'POST',
      body: JSON.stringify({
        trigger: 'course_registration',
        inputData: data,
      }),
    });
  }

  // Get transaction status
  async getTransactionStatus(transactionId: string) {
    return this.request<TransactionDetails>(
      `/api/v1/transactions/${transactionId}`
    );
  }

  // Document download URLs
  getTicketPdfUrl(ticketId: string): string {
    return `${API_BASE_URL}/api/v1/tickets/${ticketId}/pdf`;
  }

  getInvoicePdfUrl(invoiceId: string): string {
    return `${API_BASE_URL}/api/v1/invoices/${invoiceId}`;
  }
}

export const vc83Api = new VC83APIClient();
```
